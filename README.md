# CERN

[CouchDB][CouchDB], [Express][Express], [React][React] and [Node.js][Node.js] boilerplate.

## Prerequisites

- [Docker][Docker]

## Build & Run

Build and run Docker containers in the background:

```
docker-compose up -d
```

## Interfaces

- [CouchDB][CouchDB interface]
